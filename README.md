# Kare wa Kanojo Vita
A port of the indie visual novel "Kare wa Kanojo" by  Max le Fou, based on the latest public release.
<img width="1600" height="1200" alt="3725509881_preview_screenshot2" src="https://github.com/user-attachments/assets/eaac6082-e6c6-483e-a904-c2bccd368826" />

FIRST OF ALL: You'll need a modded Vita for this
This port is based on the Ren'Py 7 port by SonicMastr.

Install the .vpk with Vitashell and copy your "game" folder to ux0:app/RPYKWKFAN/
The folder already exists there, but it's empty.
Get the files from the official site: http://kwk.maxlefou.com/
or on Steam https://store.steampowered.com/app/1024790/Kare_wa_Kanojo/
The game would take a while to load but this is due to hardware limitations and how Ren'Py works on the Vita.
If you're wondering about the, now unavaliable, "Donator DLC", no. It doesn't work unfortunately. Just keep in mind it does nothing, so you won't be missing anything.
